# Israeli Document Formatting Standards

## Page Layout
- Standard page size: A4 (210mm x 297mm)
- Margins: 2.5cm all sides (standard), 2cm for dense documents
- Text direction: RTL (right-to-left) for Hebrew content
- Paragraph alignment: Right-aligned by default

## Hebrew Fonts
| Font | Style | Usage |
|------|-------|-------|
| David | Serif | Government documents, legal |
| Heebo | Sans-serif | Modern, tech, startups |
| Miriam | Serif | Academic, traditional |
| Arial Hebrew | Sans-serif | General purpose |
| Rubik | Sans-serif | Marketing, friendly |

## Date Formats
- Standard: DD/MM/YYYY (31/12/2025)
- Formal: DD בMONTH YYYY (31 בדצמבר 2025)
- Hebrew calendar: כ"ה בטבת תשפ"ו

## Address Format
```
[Street name] [number]
[City], [postal code - 7 digits]
ישראל
```

Example:
```
רחוב הרצל 42
תל אביב-יפו, 6100000
ישראל
```

## Phone Number Format
- Mobile: 05X-XXX-XXXX
- Landline: 0X-XXX-XXXX
- International: +972-X-XXX-XXXX

## Common Document Types
- מכתב רשמי (Formal Letter)
- חוזה (Contract)
- הצהרה (Declaration)
- הסכם (Agreement)
- אישור (Certificate/Confirmation)
- חשבונית מס (Tax Invoice)
- קבלה (Receipt)
- תעודת זהות העתק (ID Copy)

## Government Document Conventions
- Teudat Zehut (ID): 9-digit number with check digit
- Business number (Osek Murshe): 9-digit
- Company number (Chevra): starts with 51/52
- Non-profit (Amuta): starts with 58
