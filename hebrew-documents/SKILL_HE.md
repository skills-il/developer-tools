# מסמכים בעברית

## הנחיות

### שלב 1: הכנת סביבת Python

התקנת python-docx:

```bash
pip install python-docx
```

### שלב 2: הגדרת המסמך ל-RTL עברי

הגדרת כיוון ברירת מחדל ל-RTL עם גופנים עבריים (David, Heebo). שימוש באלמנט `w:bidi` על פסקאות ו-`w:rFonts` עם `w:cs` לגופני Complex Script.

### שלב 3: עיצוב מסמכים ישראלי

- הגדרת עמוד: A4 (21x29.7 ס"מ), שוליים 2.5 ס"מ
- תאריכים: DD/MM/YYYY או שמות חודשים עבריים
- כתובות: רחוב + מספר, עיר, מיקוד (7 ספרות)
- טלפון: פורמט 05X-XXX-XXXX

### שלב 4: תבניות נפוצות

**מכתב רשמי**: תאריך, נמען (לכבוד), נושא (הנדון), פנייה (שלום רב), גוף, סיום (בכבוד רב), חתימה.

**חוזה**: כותרת (ממורכזת), סעיף צדדים, סעיפים ממוספרים, בלוקי חתימה.

### שלב 5: טבלאות RTL

הוספת `w:bidiVisual` למאפייני הטבלה. הפיכת סדר עמודות לתצוגת RTL נכונה.

### שלב 6: מוסכמות טפסים ממשלתיים

- תעודת זהות: פורמט 9 ספרות
- נוסח הצהרה סטנדרטי
- בלוקי חתימה ותאריך
