### שלב 2: הגדרת המסמך ל-RTL עברי

יש להגדיר את כיוון ברירת המחדל של המסמך ל-RTL ולהגדיר גופנים עבריים:

**יצירת מסמך עברי חדש:**

```python
def create_hebrew_document():
    """יצירת מסמך Word חדש עם הגדרות RTL לעברית."""
    doc = Document()
    style = doc.styles['Normal']
    font = style.font
    font.name = 'David'
    font.size = Pt(12)
    # הגדרת גופן Complex Script לתווים עבריים
    rPr = style.element.get_or_add_rPr()
    cs_font = OxmlElement('w:rFonts')
    cs_font.set(qn('w:cs'), 'David')
    rPr.append(cs_font)
    return doc
```

**הגדרת כיוון פסקה מימין לשמאל:**

```python
def set_paragraph_rtl(paragraph):
    """הגדרת פסקה לכיוון טקסט מימין לשמאל."""
    pPr = paragraph._p.get_or_add_pPr()
    bidi = OxmlElement('w:bidi')
    bidi.set(qn('w:val'), '1')
    pPr.append(bidi)
    paragraph.alignment = WD_ALIGN_PARAGRAPH.RIGHT
```

### שלב 4: תבניות מסמכים ישראליים נפוצים

מבנה סטנדרטי של מכתב עסקי ישראלי:

1. כותרת החוזה (ממורכזת)
2. "הצדדים להסכם:" + פירוט הצדדים
3. סעיפים ממוספרים עם כותרות משנה
4. סעיף חתימות בתחתית

## פתרון בעיות

- **בעיה**: טקסט עברי מופיע מיושר לשמאל במסמך
  **פתרון**: יש לוודא שאלמנט `w:bidi` עם `val='1'` מוגדר על מאפייני כל פסקה. כמו כן, יש להגדיר `WD_ALIGN_PARAGRAPH.RIGHT` ליישור ימני מפורש.

- **בעיה**: עמודות טבלה בסדר הפוך (LTR במקום RTL)
  **פתרון**: יש להוסיף אלמנט `w:bidiVisual` למאפייני הטבלה. כמו כן, יש להפוך את סדר העמודות במערכי הנתונים לפני אכלוס הטבלה.

- **בעיה**: הגופן מציג תווים לטיניים במקום עברית
  **פתרון**: יש להגדיר את גופן ה-Complex Script (`w:rFonts` עם מאפיין `w:cs`) בנוסף לגופן הרגיל. עברית משתמשת במשפחת גופני Complex Script ב-Word.

- **בעיה**: רשימות ממוספרות סופרות מצד שמאל במקום ימין
  **פתרון**: יש להחיל כיוון RTL על פסקת הרשימה. ב-python-docx, יש להגדיר ידנית את מאפיין `w:bidi` על כל פסקה ממוספרת.

- **בעיה**: תוכן מעורב (עברית + אנגלית + מספרים) מוצג בסדר שגוי
  **פתרון**: אלגוריתם ה-Unicode Bidirectional מטפל ברוב המקרים אוטומטית. יש לוודא שכיוון הפסקה הבסיסי הוא RTL. למקרי קצה, ניתן להשתמש בתווי בקרה של Unicode: RLM (U+200F) ו-LRM (U+200E).
