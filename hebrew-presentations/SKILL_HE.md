# מצגות בעברית

## הנחיות

### שלב 1: הכנת סביבת Python

התקנת python-pptx:

```bash
pip install python-pptx
```

### שלב 2: הגדרת RTL ברמת המצגת

כל מצגת בעברית חייבת לכלול הגדרת כיוון RTL הן ברמת השקופית והן ברמת מסגרת הטקסט.

### שלב 3: בחירת גופנים עבריים

| גופן | מתאים ל- | סגנון |
|------|----------|-------|
| Heebo | הייטק/סטארטאפ | Sans-serif |
| Rubik | שיווק | Sans-serif מעוגל |
| David | ממשלתי/רשמי | Serif |
| Frank Ruhl Libre | אקדמי | Serif |

חשוב להגדיר את גופן ה-Complex Script (a:cs) לתווים עבריים.

### שלב 4: טיפול בטקסט דו-כיווני

שימוש ב-runs נפרדים לעברית ואנגלית. מספרים מוצגים LTR אוטומטית בפסקאות RTL.

### שלב 5: מוסכמות עסקיות ישראליות

מבנה מצגת משקיעים: פתיחה, בעיה, פתרון, גודל שוק, מודל עסקי, משיכה, צוות, תחרות, פיננסים, בקשה.

עיצוב: סימן שקל, תאריכים DD/MM/YYYY, פסיק כמפריד אלפים.
