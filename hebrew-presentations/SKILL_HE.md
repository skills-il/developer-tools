# מצגות בעברית

## הנחיות

### שלב 1: הכנת סביבת Python

התקנת python-pptx:

```bash
pip install python-pptx
```

### שלב 2: הגדרת RTL ברמת המצגת

כל מצגת בעברית חייבת לכלול הגדרת כיוון RTL הן ברמת השקופית והן ברמת מסגרת הטקסט.

### שלב 3: בחירת גופנים עבריים

| גופן | מתאים ל- | סגנון |
|------|----------|-------|
| Heebo | הייטק/סטארטאפ | Sans-serif |
| Rubik | שיווק | Sans-serif מעוגל |
| David | ממשלתי/רשמי | Serif |
| Frank Ruhl Libre | אקדמי | Serif |

חשוב להגדיר את גופן ה-Complex Script (a:cs) לתווים עבריים.

### שלב 4: טיפול בטקסט דו-כיווני

שימוש ב-runs נפרדים לעברית ואנגלית. מספרים מוצגים LTR אוטומטית בפסקאות RTL.

### שלב 5: מוסכמות עסקיות ישראליות

מבנה מצגת משקיעים: פתיחה, בעיה, פתרון, גודל שוק, מודל עסקי, משיכה, צוות, תחרות, פיננסים, בקשה.

עיצוב: סימן שקל, תאריכים DD/MM/YYYY, פסיק כמפריד אלפים.

## דוגמאות

### דוגמה 1: יצירת מצגת מכירות בעברית
המשתמש אומר: "בנה מצגת pitch deck של 10 שקופיות בעברית למשקיעים"
פעולות:
1. הגדרת PowerPoint עם כיוון RTL וגופני Heebo/David
2. יצירת שקופיות: פתיחה, בעיה, פתרון, שוק, מוצר, משיכה, צוות, פיננסים, בקשה, יצירת קשר
3. שימוש בפורמט מטבע שקלים ונתוני שוק ישראלי
4. יישור תבליטים וטקסט מימין לשמאל
תוצאה: מצגת pitch deck מקצועית בעברית עם הקשר ישראלי

### דוגמה 2: יצירת מצגת הדרכה בעברית
המשתמש אומר: "צור מצגת הדרכה בעברית לעובדים חדשים"
פעולות:
1. יצירת מצגת RTL עם מקום למיתוג חברה
2. מבנה: ברוכים הבאים, סקירת חברה, מבנה צוות, מדיניות, כלים, שאלות ותשובות
3. הוספת הערות מציג בעברית לכל שקופית
4. הוספת שקופית סדר יום עם כותרות סעיפים בעברית
תוצאה: מצגת קליטת עובדים בעברית עם הערות מציג

## משאבים מצורפים

### סקריפטים
- `scripts/pptx_rtl_fixer.py` -- מאמת ומתקן יישור RTL בקבצי PowerPoint. הרצה: `python scripts/pptx_rtl_fixer.py --help`

### חומרי עזר
- `references/israeli-business-slides.md` -- מוסכמות מצגות עסקיות ישראליות כולל מבני שקופיות נפוצים, טיפוגרפיה עברית לשקופיות ונורמות תרבותיות. יש לעיין בו בעת בניית מצגות עסקיות או למשקיעים לקהל ישראלי.

## פתרון בעיות

### שגיאה: "תבליטים מופיעים בצד הלא נכון"
סיבה: PowerPoint מוגדר כברירת מחדל ליישור תבליטים LTR
פתרון: יש להגדיר יישור פסקה ל-RIGHT ולהפעיל כיוון פסקה RTL. ב-`python-pptx`, יש להגדיר `paragraph.alignment = PP_ALIGN.RIGHT`.

### שגיאה: "תרשימים מציגים תוויות עברית בצורה שגויה"
סיבה: מסגרות טקסט בתרשימים מוגדרות כברירת מחדל ל-LTR
פתרון: לאחר יצירת נתוני התרשים, יש לעבור על כל מסגרות הטקסט בתרשים ולהגדיר את כיוון הפסקה ל-RTL. תוויות ציר בעברית עשויות לדרוש הקצאת גופן ידנית.
