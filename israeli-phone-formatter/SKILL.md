# Israeli Phone Formatter

## Instructions

### Step 1: Identify Phone Number Type
Check prefix to determine type:
- Mobile: 050-058
- Landline: 02-09
- VoIP: 072-077

### Step 2: Validate
Validate the data before proceeding.

### Step 3: Format
Convert between local and international formats.

## Examples

### Example 1: Validate number
User says: "Check 052-1234567"
Result: Valid mobile number.
