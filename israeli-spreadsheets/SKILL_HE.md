# גיליונות ישראליים

## הנחיות

### שלב 1: הכנת סביבת Python

התקנת openpyxl:

```bash
pip install openpyxl
```

### שלב 2: קבועים פיננסיים ישראליים

- שיעור מע"מ: 18% (2025)
- מדרגות מס 2025: 10% עד 84,120 ש"ח, 14% עד 120,720, 20% עד 193,800, 31% עד 269,280, 35% עד 560,280, 47% עד 721,560, 50% מעל
- ערך נקודת זיכוי: 2,904 ש"ח/שנה, 2.25 נקודות לתושבים
- ביטוח לאומי: 0.4% (נמוך) / 7% (גבוה)
- מס בריאות: 3.1% (נמוך) / 5% (גבוה)

### שלב 3: פונקציות חישוב מס

חישוב מס פרוגרסיבי (שולי) עם ניכוי נקודות זיכוי. שימוש ב-Decimal של Python לדיוק.

### שלב 4: חוברות עבודה מעוצבות

הגדרת `ws.sheet_view.rightToLeft = True` לגיליונות RTL. פורמט שקלים: '#,##0.00 "₪"'

### שלב 5: תבניות

- **חשבונית מס**: פרטי עסק/לקוח, טבלת פריטים, סכום ביניים, מע"מ 18%, סה"כ
- **תלוש משכורת**: תשלומים, ניכויים (מס הכנסה, ביטוח לאומי, מס בריאות, פנסיה, קרן השתלמות), שכר נטו
- **מחשבון ארנונה**: תעריפים לפי עיר (תל אביב 55.80, ירושלים 40.50, חיפה 33.20, באר שבע 27.90, נתניה 43.10 למ"ר/דו-חודשי)

## דוגמאות

### דוגמה 1: יצירת מחשבון שכר ישראלי
המשתמש אומר: "בנה גיליון Excel לחישוב שכר לעובד ישראלי"
פעולות:
1. יצירת חוברת עבודה RTL עם כותרות בעברית
2. הוספת מדרגות מס הכנסה (שיעורי 2024: 10%-50%)
3. חישוב ביטוח לאומי (ספי 3.5%/12%), מס בריאות (3.1%/5%)
4. הוספת פנסיה (6.5% עובד + 6.5% מעסיק) וקרן השתלמות
5. עיצוב כל הסכומים בשקלים עם תוויות בעברית
תוצאה: מחשבון שכר ישראלי מלא עם חישוב שכר נטו

### דוגמה 2: יצירת תבנית חשבונית ישראלית
המשתמש אומר: "צור תבנית חשבונית מס בעברית עם חישוב מע"מ"
פעולות:
1. הגדרת Excel עם כיוון RTL וכותרות עמודות בעברית
2. הוספת שדות פרטי עסק (מספר עוסק מורשה, כתובת)
3. הוספת שורות פריטים עם כמות, מחיר ליחידה, סכום ביניים
4. חישוב מע"מ 18%, הצגת סה"כ בשקלים
5. הוספת שדות מספר חשבונית ותאריך עברי
תוצאה: תבנית חשבונית עם מע"מ בעברית

## משאבים מצורפים

### סקריפטים
- `scripts/shekel_formatter.py` -- מעצב תאי Excel עם מוסכמות מטבע, תאריך ומספרים ישראליים. הרצה: `python scripts/shekel_formatter.py --help`

### חומרי עזר
- `references/israeli-tax-rates.md` -- מדרגות מס ישראליות עדכניות, שיעורי ביטוח לאומי, שיעור מע"מ, דרישות פנסיה וקבועים פיננסיים נפוצים. יש לעיין בו בעת בניית חישובים פיננסיים בהקשר ישראלי.

## פתרון בעיות

### שגיאה: "סימן השקל מופיע בצד הלא נכון של המספר"
סיבה: הגדרת Locale של Excel לא מוגדרת לעברית/ישראל
פתרון: יש להשתמש במחרוזת פורמט `#,##0.00 ₪` (סימן אחרי המספר) בהתאם למוסכמה הישראלית, או להגדיר את ה-locale של חוברת העבודה ל-he-IL.

### שגיאה: "כותרות עמודות בעברית מוצגות כסימני שאלה"
סיבה: חוברת העבודה לא נשמרה בקידוד UTF-8 או שהגופן לא תומך בעברית
פתרון: יש לוודא שחוברת העבודה משתמשת בגופן תואם Unicode (כגון David, Heebo או Arial). בעת שימוש ב-openpyxl, מחרוזות עברית מקודדות אוטומטית ב-UTF-8.
